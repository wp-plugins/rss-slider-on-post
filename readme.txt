RSS slider on post is a small WordPress plugin to create the scroller/slider text gallery into the posts and pages, that makes rss integration to your web site very easy. In the admin we have option to add the rss feed link. It will create the text slider using the given RSS feed. In the front end it will scroll the rss title with link and rss description.

**Features of this plugin**

* Simple installation and customization.
* Option to add any rss feed.
* Easy scroller setting.

**Plug-in configure**

First install and activate the plugin, after use the below short code in the posts and pages.
